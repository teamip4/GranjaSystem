Sistema de Gerenciamento de Granjas de Postura (Avicultura)

Descrição
Este projeto consiste no desenvolvimento de um Sistema Integrado para
Gerenciamento e Monitoramento da Produção de Ovos em Granjas de Postura.
A solução visa digitalizar o controle de produção avícola, substituindo anotações
manuais por uma plataforma que centraliza registros de produção de ovos, consumo de ração,
mortalidade e gestão financeira. O sistema fornece inteligência de dados para o produtor, com
alertas automáticos sobre o ciclo de vida das aves e relatórios de desempenho.


Objetivos do projeto:
● Monitoramento produtivo: Acompanhar diariamente a coleta de ovos e identificar
automaticamente o Início de Postura (>5%) e o Pico de Postura (>90%);
● Controle nutricional: Calcular e registrar o consumo diário de ração (média de
110-115g/ave) para evitar desperdícios;
● Controle sanitário: Monitorar taxas de mortalidade e registrar vacinação, emitindo
alertas em caso de mortalidade em larga escala.
● Rotatividade de lotes: Gerenciar a rotatividade de lotes, alertando o momento ideal
de alojamento (ex: a cada 26 semanas);
● Gestão financeira: Relatórios detalhados de gastos com insumos (ração/vacinas)
versus lucro com venda de ovos;
